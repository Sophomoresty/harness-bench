# Company denies massive breach — only small test dataset impacted

**Source: NewsSite B — 2026-03-16 11:05**

Responding to a rival outlet’s “hundreds of millions leaked” headline, NewsSite B obtained an internal memo stating **there was no large-scale real-user data breach**.

Security teams found abnormal access to an internal **test environment** during routine audits; roughly **5,000 test accounts** may have been exposed. These accounts contain **no real user information**.

“We patched immediately and notified regulators. There is no evidence real user data was accessed,” a spokesperson said.

**Key claims:**
- Discovery date: March 15
- Affected records: ~5,000 test accounts
- Data types: test accounts only (not production users)

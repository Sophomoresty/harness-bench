# Official statement

**Published: 2026-03-16 14:00**

We are aware of recent media coverage about a data-security incident. User privacy and security are priorities; we state the following:

1. **Overview:** During a routine security audit on **2026-03-14**, we found unauthorized access to an **internal test environment** containing only fictional data used for QA. **No production environment containing real personal information was accessed.**

2. **Scope:** Comprehensive review shows **1,247 test accounts** affected. **No real user data was exfiltrated.**

3. **Data involved:** Only test email addresses (format `test*@example.com`) and intentionally weak test passwords (e.g. `Password123`). **No names, phone numbers, addresses, or payment information.**

4. **Actions taken:**
   - Isolated and patched the vulnerable path
   - Notified relevant regulators
   - Increased internal monitoring

5. **User guidance:** Production user data remains secure. We still recommend periodic password rotation and enabling two-factor authentication.

We will continue transparent updates.

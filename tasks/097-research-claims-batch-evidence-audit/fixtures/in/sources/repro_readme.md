# Reproducibility Package README

The deposited package contains `results_table.csv`, this README, and notes. It does not contain `raw/events.parquet`, `analysis_config.yaml`, or `results/subgroup_ablation.csv`.

The main driver `scripts/run_analysis.py` is referenced in the paper but is absent from the deposit.

Without the raw event bundle and analysis config, the headline estimates cannot be rerun end-to-end from the shipped package.

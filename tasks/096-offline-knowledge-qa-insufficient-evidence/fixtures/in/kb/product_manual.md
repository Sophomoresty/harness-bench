# Product Manual

Section PM-1 Vault Sync Snapshots

Vault Sync stores workspace snapshots using AES-256-GCM encryption at rest. Snapshot metadata is indexed separately from document payloads.

Section PM-2 Recovery

Rivergate workspaces support two recovery channels: administrator self-service restore and support-assisted restore. The manual does not define a phone escalation channel.

Section PM-3 Plans

SSO is available on Enterprise plans. Basic plan customers do not have SSO access.

Section PM-4 Customer-Managed Keys

Customer-managed keys are owned by the Security Platform team. The manual says key rotation is "periodic" but does not state an exact interval.

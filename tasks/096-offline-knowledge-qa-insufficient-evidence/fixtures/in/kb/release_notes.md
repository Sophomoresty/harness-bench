# Release Notes

Release 3.4

RN-3.4-Export: Vault Export jobs now allow attachments up to 200 MB per item. Older 100 MB limits are retired.

Release 3.5

RN-3.5-Delta: Delta Indexing now skips unchanged binary attachments and records a checksum marker for skipped files.

RN-3.5-Regions: Vault Sync private beta is available in us-east-1 and eu-central-1. No Asia-Pacific region is listed.

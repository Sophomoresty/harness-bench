# Document Validity Rules

## DOC-1 Current government ID
government_id must be unexpired as of 2026-05-04.

## DOC-2 Address match
proof_of_address must match the address on the customer profile.

## DOC-3 Business registration
business_registration must include a registration number and current status.

## DOC-4 Beneficial owner attestation
beneficial_owner_attestation must list all beneficial owners with ownership percentages. For this benchmark, missing percentage data makes the document invalid.

## DOC-5 Tax form
tax_form must be signed.

## DOC-6 Source of funds
source_of_funds_statement is required for nonprofit customers and must name a source category.

# Experiment analysis plan

- Control variant: A
- Treatment variant: B
- Primary metric: purchase conversion within 7 days of first exposure.
- Statistical test: two-proportion z-test, two-sided, alpha 0.05.
- Exclude employees, bot users, duplicate assignments, conversions before exposure, and conversions outside the 7-day window.
- Mobile cohort is directional only because the sample is underpowered.
